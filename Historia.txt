revision primera parte de git 
crear archivos con touch
crear carpetas con mkdir editar texto plano con code "nombre del archivo y extension"

para agregar un archivo debo usar git add nombre del archivo y su extencion por ejemplo
git add historia.txt

si son varios archivos puedo utilizar git add .

luego para subirlo a git de utilizar el comando commit por ejemplo
git commit -m "una descripcion de lo que se sube"

(la m- es para indicar un mensaje al subir una actualizacion, es una buena practica y por eso lo aplicaremos)
